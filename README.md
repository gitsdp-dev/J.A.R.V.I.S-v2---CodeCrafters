# 🚀 What is J.A.R.V.I.S v2?

J.A.R.V.I.S v2 is the next evolution of the original J.A.R.V.I.S project.

Built completely in Python, it transforms your computer into an intelligent AI workstation capable of understanding natural language, interacting with your operating system, analyzing documents, understanding your screen, and executing multi-step workflows—all through one seamless interface.

Rather than functioning as a simple chatbot, J.A.R.V.I.S v2 acts as an intelligent operating companion that bridges the gap between human conversation and computer interaction.

# 🛠️ Installation

## Clone the Repository

```bash
git clone https://github.com/gitsdp-dev/J.A.R.V.I.S-v2---CodeCrafters.git
```

## Navigate into the Project

```bash
cd J.A.R.V.I.S-v2---CodeCrafters
```

## Install Dependencies

```bash
pip install -r requirements.txt
playwright install
```

## Start J.A.R.V.I.S

```bash
python main.py
```

During the first launch, J.A.R.V.I.S will guide you through the initial configuration process, including setting up your Gemini API key.
